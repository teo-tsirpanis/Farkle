Namespace: Farkle

Some useful functions and types that could be used from many points from the library.

Nested types and modules

TypeDescription
Indexable<'a>

Anything that can be indexed. The type is just a record with a value and an index.
Indexed<'a>

A type-safe reference to a value based on its index.
Position

A point in 2D space with integer coordinates, suitable for the position of a character in a text.
RangeSet<'a>

A simple and efficient set of items based on ranges. Instead of storing all the elements of a set, only the first and last are. This closely follows the paradigm of GOLD Parser 5 sets.
ModuleDescription
Indexable

Functions for working with Indexable<'a>.
Indexed

Functions for working with Indexed<'a>.
List

Some more utilities to work with lists.
Position

Functions to work with the Position type.
RangeSet

Functions to work with the RangeSet type.
String

Some utilities to work with strings
Trial

Functions to work with the Chessie.ErrorHandling.Result type. I will propably make a PR to add them in the future.

mustBeSome x
Signature: x:'a option -> 'a
Type parameters: 'a

Returns the value of an Option. Raises an exception if the option was None. Are you inside a State monad and don't want to spill your code with StateResult? Are you definitely sure that your Option does really contain a value, but the type system disagrees? In this case, you should use me! But use me carefully and very sparingly. That thing is like unsafePerformIO, but fortunately, not-so-destructive.
